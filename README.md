# 🏫 Kids Ministry App - Iglesia Emmanuel
> **Desarrollado por AM.DEV (Antonio Montes de Oca Torres)**

Una Aplicación Web Progresiva (PWA) diseñada para optimizar la planificación, asignación de roles y gestión de maestros en el ministerio infantil. 

---

## ✨ Características Principales
* **📱 Experiencia PWA:** Instalable en dispositivos móviles (Android/iOS) con funcionamiento offline.
* **🗓️ Gestión de Roles:** Calendario interactivo para asignar maestros a diferentes servicios y grupos.
* **👥 Control de Maestros:** Base de datos en tiempo real para gestionar el equipo de voluntarios.
* **📄 Exportación Inteligente:** Generación de formatos optimizados para impresión y compartición rápida vía WhatsApp.
* **💎 Glass Design:** Interfaz moderna, limpia y minimalista basada en efectos de transparencia y desenfoque.

## 🛠️ Tecnologías Utilizadas
* **React 18** - Biblioteca principal para la interfaz de usuario.
* **Vite** - Herramienta de construcción ultra rápida.
* **Tailwind CSS** - Estilizado mediante utilidades y diseño responsivo.
* **Supabase** - Backend as a Service (BaaS) para base de datos y autenticación en tiempo real.
* **Lucide React** - Set de iconos vectoriales consistentes.
* **Vite PWA Plugin** - Configuración de Service Workers y Manifiesto.

## 🚀 Instalación y Configuración
1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/tonnyweb/kids-ministry-app.git](https://github.com/tonnyweb/kids-ministry-app.git)
    ```
2.  **Instalar dependencias:**
    ```bash
    npm install
    ```
3.  **Configurar variables de entorno:**
    Crea un archivo `.env` con tus credenciales de Supabase:
    ```env
    VITE_SUPABASE_URL=tu_url_aqui
    VITE_SUPABASE_ANON_KEY=tu_clave_aqui
    ```
4.  **Correr en modo desarrollo:**
    ```bash
    npm run dev
    ```
